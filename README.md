A collection of shaders adapted from Ben Cloward's 'Shader Graph Basics' series.

Each 'ep' folder corresponds to an episode from the playlist:
    https://www.youtube.com/playlist?list=PL78XDi0TS4lEBWa2Hpzg2SRC5njCcKydl

Each ep contains meshes with already applied shaders, and every shader has both
a code and node (visual shader) version. Toggle the visibility of nodes in the
scene tree to see the shader/mesh of your choice.
